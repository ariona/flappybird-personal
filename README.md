# Flappy Bird 🐦

A simple but fun Flappy Bird clone built with HTML5 Canvas.

## 🎮 How to Play

- **PC:** Click anywhere or press **Space** to jump
- **Mobile:** Tap the canvas to jump

Aim to fly through the gaps between the pipes without hitting them or the ground!

## 🕹️ Features

- Smooth physics-based movement
- Randomly generated pipes
- Score tracking
- Game over screen with replay option
- Responsive design
- Beautiful graphics with gradients and animations

## 🛠️ Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
